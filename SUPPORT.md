# Support & Help

Need help with NETRA? Here's where to get support:

## Bug Reports & Feature Requests

Found a bug or have a feature idea? Open an issue on GitHub:

**GitHub Issues**: https://github.com/netra/netra/issues

Include:
- Version number (`netra --version`)
- Steps to reproduce
- Expected vs. actual behavior
- Environment details (OS, Python version, etc.)

## Questions & Discussions

Have a question about how to use NETRA? Use GitHub Discussions:

**GitHub Discussions**: https://github.com/netra/netra/discussions

This is the best place for:
- How-to questions
- Architecture discussions
- Best practices
- Integration help

## Documentation

Full documentation is available at:

- **README**: `/README.md` — Getting started guide
- **docs/ directory**: In-depth guides and API reference
- **CLAUDE.md**: Development guidelines and project structure
- **CONTRIBUTING.md**: How to contribute to NETRA

## Security Vulnerabilities

Found a security issue? **Do not open a public GitHub issue.**

See **SECURITY.md** for responsible disclosure instructions.

For commercial support inquiries, consulting, or enterprise deployments, contact the maintainer.
